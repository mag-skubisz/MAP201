# MAP201 TPs

This repository contains coursework for MAP201 - Découverte des mathématiques appliquées, organized into two main tracks:

- **TP_EDO_MAP201**: Ordinary Differential Equations (EDO) practical sessions and exported reports.
- **TP_IMAGE_MAP201**: Image processing practical sessions, utility scripts, and exported reports.

## Project Structure

- **TP_EDO_MAP201/**
  - `TP-EDO-1` to `TP-EDO-5`
  - Jupyter notebooks (`.ipynb`) for each TP
  - HTML exports of completed work

- **TP_IMAGE_MAP201/**
  - `TP0` to `TP5`
  - Jupyter notebooks (`.ipynb`) and supporting Python scripts (`utils.py`, exercise scripts)
  - HTML exports of completed work
